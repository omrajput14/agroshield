# 🌪️ AgroShield AI

**AgroShield AI** is an advanced, real-time IoT monitoring and predictive analytics platform built specifically for banana farmers to protect their crops against severe winds, cyclones, and adverse weather conditions. 

By combining live sensor telemetry with a Scikit-Learn Machine Learning engine, AgroShield AI proactively predicts crop damage probability and automatically deploys windbreak nets to save yields before storms hit.

## ✨ Key Features

### 🧠 Real-Time AI Prediction Engine
- Features a **Random Forest Regressor** Machine Learning model trained on synthetic historical storm data.
- Continuously ingests live sensor data (wind speed, gusts, temperature, humidity) and farm metadata (area, nets deployed) to output a dynamic **Damage Probability Score**.
- Generates intelligent mitigation recommendations on the fly.

### 🌬️ Automated Risk Engine
- Evaluates live telemetry against predefined safety thresholds.
- **Auto-Deploys** physical windbreak nets via IoT when dangerous wind gusts are detected.
- Escalates risk levels from Safe -> Moderate -> High -> Severe dynamically.

### 📱 SMS Notification Alert System
- Fully integrated alerting system.
- Automatically dispatches an SMS payload to the farm owner's phone when nets are auto-deployed or cyclonic winds are detected.
- Built-in Mock Mode for development, ready to plug into **Twilio** for production.

### 📊 Stunning Real-Time Dashboard
- **React + Vite** frontend styled with a modern, glassmorphism dark-mode aesthetic using **Tailwind CSS**.
- Real-time data jittering to simulate live IoT WebSocket connections.
- Interactive **Recharts** displaying 24-hour dual-line wind and gust trends.

### 🔐 Role-Based Access Control
- JWT-based authentication system supporting multiple user tiers: **Admin**, **Farm Manager**, and **Farmer**.

## 🛠️ Technology Stack

**Frontend:**

## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/omrajput14/agroshield.git
cd agroshield
```

### 2. Backend Setup
Navigate to the backend directory, install dependencies, and seed the database.
```bash
cd backend
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Seed the SQLite database with mock users and farms
python app/seed.py

# Train the Machine Learning model
python scripts/train_ml_model.py

# Start the FastAPI server
uvicorn app.main:app --reload
```

### 3. Frontend Setup
Open a new terminal and navigate to the root directory.
```bash
npm install
npm run dev
```

### 4. Access the Application
Open your browser and navigate to `http://localhost:5173`. 
Log in with the mock credentials generated by the seed script:
- **Email:** farmer@agroshield.ai
- **Password:** password123

## 🚨 Triggering a Mock Storm
Want to see the auto-deployment and SMS notification system in action? While the backend server is running, open a new terminal and run:
```bash
cd backend
source venv/bin/activate
python scripts/trigger_alert.py
```
This will simulate a Category 1 cyclonic wind spike at Sakshi Agro Farm, auto-deploy the nets, and dispatch a mock SMS to the terminal!
